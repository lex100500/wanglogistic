# WangLogistic - Обменник валют

## Суть
Telegram-бот + веб-дашборд для обмена валют. Клиент создаёт заявку через бота, менеджеры получают уведомление и берут заказ в работу. Владелец управляет всем через веб-панель.

## Telegram-бот

### Клиент
- `/start` — приветствие + меню
- "Обменять" — выбор валютной пары (USD/RUB, EUR/RUB, USDT/RUB и т.д.)
- Ввод суммы → показ курса → подтверждение
- Создание заявки → уникальная ссылка
- Статус заявки — отслеживание в реальном времени
- История обменов

### Менеджер
- Уведомление о новой заявке (в группу или лично)
- Кнопка "Взять заказ" — закрепляется за менеджером
- Чат с клиентом через бота (relay)
- Смена статуса: принято → в работе → завершено / отменено

### Статусы заявки
`new` → `taken` → `in_progress` → `completed` / `cancelled`

## Веб-дашборд (для владельца)

### Страницы
1. **Обзор** — кол-во заявок за сегодня/неделю/месяц, объём, конверсия
2. **Заявки** — таблица всех заявок с фильтрами (статус, дата, менеджер, валюта)
3. **Менеджеры** — список, статистика по каждому, добавить/удалить
4. **Курсы** — установка курсов валютных пар, маржа
5. **Настройки** — токен бота, группа менеджеров, тексты сообщений

## Стек
- **Бот**: Python 3.12 + aiogram 3
- **БД**: SQLite (tasks.db стиль — просто и быстро)
- **Дашборд**: Express.js + vanilla frontend (один index.html)
- **Деплой**: systemd сервисы на текущем сервере
