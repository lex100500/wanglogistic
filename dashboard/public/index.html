📌 Переменные курса
Используй {тир:N} — подставляет итоговый курс для тира с порогом N юаней из таблицы ниже.
Число порога пиши обычным текстом, переменная только для курса.
{курс} актуальный курс RUB/CNY
{тир:400} курс − скидка тира 400 CNY
{тир:2000} курс − скидка тира 2000 CNY
{тир:8000} курс − скидка тира 8000 CNY
{банк:Т-Банк} размер скидки банка Т-Банк
{банк:СБЕР} размер скидки банка СБЕР
✅ От 400 CNY — {тир:400} руб
✅ С Т-Банком — {банк:Т-Банк} руб
❌ От {тир:400} CNY — {тир:400} руб (оба заменятся курсом)
⚡ При изменении скидки в таблице тиров — курс в акциях обновляется автоматически.
Скидка от суммы в CNY. Стакается с Т-Банком. Тиры сортируются автоматически.
От (CNY) Скидка к курсу
0 = без ограничений
0 = без ограничений
Стакается с объёмными скидками. 0 = без скидки.

Автоматическая отправка/редактирование сообщения с курсом в Telegram-чатах при изменении курса.

📌 Переменные
{курс}курс покупки

📌 Как добавить чат или топик
1. Открой нужный чат/канал в Telegram → скопируй ссылку на любое сообщение
2. Вставь ссылку в поле ниже — Chat ID и Топик ID заполнятся автоматически
https://t.me/c/3734813630/2
→ Chat ID: -1003734813630   Топик: 2
Для обычного канала/группы: просто введи @username или числовой ID.
⚠️ Бот должен быть администратором чата.
